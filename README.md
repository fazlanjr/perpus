## clone project

```
git clone https://github.com/jefyokta/perpus.git
```
## instal PHP Packages via composer

```
composer install
```

## install node package

```
npm install
```
or
```
bun install
```
## make .env file, example is available in root dir

## run

```
npm run perpus
```

or

```
php artisan serve
```
